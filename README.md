# Web TASKS

WEB to management tasks

## Installation

Installation to development environment

```
# clone repo
git clone https://github.com/YahikoDev/web-tasks.git
```

```
# Install dependencies
npm i
```
```
# Create .env file
cp .env.example .env
```
```
# Config .env variables 
VITE_APP_BASE_URL=your_api_path
```
